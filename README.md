# Disaster Alleviation Foundation - ASP.NET Core Web Application

This is a secure, scalable web application to support disaster reporting, resource donations, and volunteer coordination. It is built with ASP.NET Core 7, Entity Framework Core, and integrates Azure Active Directory (Azure AD) for authentication.

---

## Features

- User authentication and registration via Azure Active Directory (Azure AD)
- Submit and manage disaster reports (location, type, description, date)
- Donation management (resource type, quantity, notes)
- Volunteer task assignment and tracking
- Responsive, Bootstrap-based UI for ease of use

---

## Prerequisites

- [.NET 7 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/7.0)
- Visual Studio 2022 or higher with ASP.NET and web development workloads
- SQL Server or LocalDB instance (LocalDB is included with Visual Studio)
- Azure Active Directory tenant and app registration
